# Data Structures and Algorithms (DSA) in C++

Welcome to my DSA journey! This repository is where I practice and track my progress as I learn and solve DSA problems in C++.

## 📌 Goals

- Improve problem-solving skills
- Prepare for coding interviews
- Build strong foundations in DSA concepts
- Stay consistent and track progress

## 📁 Repository Structure

| Folder | Description |
|--------|-------------|
| `ARRAYS/` | Array problems and techniques |
| `Arrays1/` | Additional array implementations and ADTs |
| `Dynamic memory allocation.cpp/` | Dynamic memory allocation examples (folder name includes `.cpp` suffix) |
| `LEETCODE AND GEEKS FOR GEEKS/` | Solutions to LeetCode and GFG problems |
| `Monotonic stack/` | Monotonic stack patterns |
| `POINTERS/` | Pointer concepts and examples |
| `QUEUE/` | Queue data structure implementations |
| `RECURSION/` | Recursive problem solutions |
| `SORTING/` | Sorting algorithms and related problems |
| `Sortings/` | Core sorting algorithm implementations |
| `STACK/` | Stack data structure implementations |
| `STRINGS/` | String manipulation problems |

## 🧮 Topics Covered

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort

### Arrays
- Binary Search (variants: LC704, LC33, LC34, LC852, LC153)
- 2D Arrays (spiral form, rotation, diagonal sum)
- Prefix/Suffix arrays
- Kadane's Algorithm
- Sliding Window

### Strings
- KMP Algorithm
- String matching (brute force)
- Palindrome checks

### Data Structures
- Stack (with STL and custom implementations)
- Queue (circular, using stacks)
- Monotonic Stack patterns

### Recursion
- Fibonacci, Factorial
- Power functions
- Search with recursion
