# DSA - Data Structures and Algorithms

A collection of Data Structures and Algorithms implemented in C++.

## Topics Covered

### Arrays
- Binary search (including LeetCode 33, 34, 704, 852)
- 2D array operations (rotation, spiral form, search)
- Reverse, rotate, and other array manipulations
- Fibonacci series, second largest element, prefix sums

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort

### Strings
- KMP (Knuth-Morris-Pratt) Algorithm
- Palindrome checks
- String matching (brute force)
- Kadane's Algorithm
- Longest palindrome, panagram, and more

### Stack
- Stack implementation and operations
- Valid parentheses (LeetCode 20, 921)
- Next Greater Element
- Celebrity problem
- Min stack, reverse stack

### Queue
- Queue ADT
- Circular queue
- Dequeue
- Queue using stack
- Sliding window problems

### Recursion
- Factorial, Fibonacci, GCD
- Array operations (search, reverse, sum)
- String operations (palindrome, vowel check)
- Power functions

### Pointers
- Pass by pointer and reference
- Pointers with arrays
- Double and triple pointers

### Dynamic Memory Allocation
- 1D, 2D, and 3D array dynamic allocation

### Monotonic Stack
- Next greater element
- Next smallest element
- Smallest element on left

### LeetCode & GeeksForGeeks Solutions
- Binary search problems (LC 33, 34, 35, 153, 704, 852)
- Two-pointer problems (3Sum, 4Sum)
- Trapping Rain Water (LC 42)
- KMP Algorithm (LC 28, GFG)
- Fibonacci (LC 509), Climbing Stairs (LC 70)
- Reverse String (LC 344)
- Aggressive Cows, Book Allocation, Painter Partition

## Language

All implementations are in **C++**.
